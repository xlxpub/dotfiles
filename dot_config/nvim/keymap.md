# Neovim 快捷键速查表

> `<leader>` = **空格键**｜✦ 标记为自定义/增强

---

## 一、基础移动（原生）

| 快捷键 | 说明 |
|--------|------|
| `h j k l` | 左/下/上/右 |
| `w W b B e E` | 按单词跳转（W/B/E 按空白分词） |
| `0 ^ $` | 行首（含空白）/ 行首（非空白）/ 行尾 |
| `gg G` | 文件首 / 文件尾 |
| `{ }` | 上/下一个空行段落 |
| `%` | 跳转到匹配括号 |
| `f{c} F{c}` | 行内正向/反向跳到字符 `c` |
| `t{c} T{c}` | 行内正向/反向跳到字符 `c` 前 |
| `; ,` | 重复/反向 f/t 跳转 |
| `H M L` | 屏幕顶部/中部/底部 |
| `zz zt zb` | 当前行居中/置顶/置底 |

## 二、翻页 & 滚动（含自定义）

| 快捷键 | 说明 |
|--------|------|
| `<C-d>` | 下翻半屏 + 居中 ✦ |
| `<C-u>` | 上翻半屏 + 居中 ✦ |
| `<C-f> <C-b>` | 下/上翻整屏 |
| `<C-e> <C-y>` | 屏幕向下/上滚一行（光标不动） |

## 三、搜索 & 跳转（含自定义）

| 快捷键 | 说明 |
|--------|------|
| `/pattern` `?pattern` | 向下/向上搜索 |
| `n N` | 下/上一个匹配 + 居中 ✦ |
| `* #` | 搜索光标下的词（正/反向） |
| `<Esc>` | 清除搜索高亮 ✦ |
| `s` | Flash 跳转（输入字符快速定位） ✦ |

## 四、编辑操作（原生）

| 快捷键 | 说明 |
|--------|------|
| `i a I A` | 插入（前/后/行首/行尾） |
| `o O` | 在下/上方新行插入 |
| `d{motion}` `dd` | 删除 / 删除整行 |
| `c{motion}` `cc` | 修改（删除+进入插入）/ 修改整行 |
| `y{motion}` `yy` | 复制 / 复制整行 |
| `p P` | 粘贴到后/前（yanky 增强，`<C-p>/<C-n>` 切换历史）✦ |
| `u <C-r>` | 撤销 / 重做 |
| `x X` | 删除光标下/前字符 |
| `.` | 重复上次操作 |
| `~` | 切换大小写 |
| `J` | 合并下一行到当前行 |
| `r{c}` | 替换单个字符 |
| `>< >><<` | 缩进/反缩进（选中/整行） |
| `=` | 自动缩进 |
