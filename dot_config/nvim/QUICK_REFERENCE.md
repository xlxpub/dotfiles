# 🎯 Neovim 快速参考卡片

## 🚀 快速启动

```bash
# 使用别名快速打开
v <filename>
vi <filename>
vim <filename>
nvim <filename>

# 首次使用需要安装 LSP 服务器
nvim
# 然后在 Neovim 中执行：
:Mason
# 搜索 gopls 和 lua_ls，按 'i' 安装
```

## ⌨️ 常用快捷键

### 基础操作
| 快捷键 | 功能 |
|--------|------|
| `:w` | 保存 |
| `:q` | 退出 |
| `u` / `<C-r>` | 撤销 / 重做 |
| `/pattern` | 搜索 |
| `n` / `N` | 下一个 / 上一个搜索结果 |

### 编辑快捷键（Leader = Space）
| 快捷键 | 功能 |
|--------|------|
| `<Space>w` | 保存 |
| `<Space>q` | 退出 |
| `<Space>Q` | 强制退出全部 |
| `<Esc>` | 清除搜索高亮 |

### 窗口和 Buffer
| 快捷键 | 功能 |
|--------|------|
| `<C-h/j/k/l>` | 切换窗口（左下上右） |
| `<C-Up/Down/Left/Right>` | 调整窗口大小 |
| `<S-h>` | 上一个 buffer |
| `<S-l>` | 下一个 buffer |

## 🔧 常用命令

### 在命令行中输入
```vim
" 打开资源管理器
:Oil

" 打开 Mason（LSP 管理）
:Mason

" 检查 LSP 状态
:LspInfo

" 检查 TreeSitter 状态
:TSInstall <language>

" 查看 Lazy 插件管理器
:Lazy

" 格式化当前文件
:Format

" 打开 Trouble（诊断面板）
:Trouble
```

## 📋 TextObjects（与运算符组合）

| 快捷键 | 含义 |
|--------|------|
| `af` | 整个函数 |
| `if` | 函数体 |
| `ac` | 整个类 |
| `ic` | 类体 |
| `aa` | 参数 |
| `ia` | 参数内部 |

### 示例
```vim
daf   " 删除整个函数
yif   " 复制函数体
cac   " 修改整个类
```

## 🎨 主题和显示

### 状态栏
- 左：当前模式
- 中：文件信息和诊断
- 右：编码、格式、文件类型、光标位置

### 诊断符号
- `●` - 错误
- `●` - 警告
- `󰠠` - 提示
- `●` - 信息

## 📂 配置文件位置

```
~/.config/nvim/
├── init.lua                   # 启动入口
├── lua/
│   ├── options.lua            # 编辑器设置
│   ├── keymaps.lua            # 按键绑定
│   ├── lazy-bootstrap.lua     # Lazy 初始化
│   └── plugins/               # 插件配置
└── lazy-lock.json             # 插件版本锁定
```

## 💡 提示

1. **按键提示**：按 `<Space>` 会显示当前可用的快捷键
2. **帮助文档**：在 Neovim 中输入 `:help <topic>`
3. **配置修改**：修改后需要重启 Neovim 或运行 `:source %`
4. **LSP 问题**：运行 `:LspInfo` 检查 LSP 连接状态
