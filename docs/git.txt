⏺ Git 快捷键完整整理

  全局快捷键（任何 buffer 中可用）

  ┌────────────┬──────────┬──────────────────────────┐
  │   快捷键   │   来源   │           功能           │
  ├────────────┼──────────┼──────────────────────────┤
  │ ]h         │ gitsigns │ 跳到下一个变更块         │
  ├────────────┼──────────┼──────────────────────────┤
  │ [h         │ gitsigns │ 跳到上一个变更块         │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gs │ gitsigns │ 暂存当前 hunk            │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gr │ gitsigns │ 重置当前 hunk（restore） │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gp │ gitsigns │ 预览当前 hunk 改动       │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gb │ gitsigns │ 查看行 blame             │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gd │ diffview │ 打开 Diffview            │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gc │ diffview │ 关闭 Diffview            │
  ├────────────┼──────────┼──────────────────────────┤
  │ <leader>gf │ diffview │ 当前文件 Git 历史        │
  └────────────┴──────────┴──────────────────────────┘

  ---
  Diffview — Diff 视图窗口（自定义）

  ┌────────┬─────────────────────────────────────────┐
  │ 快捷键 │                  功能                   │
  ├────────┼─────────────────────────────────────────┤
  │ q      │ 关闭 Diffview                           │
  ├────────┼─────────────────────────────────────────┤
  │ do     │ 获取对面的改动（diff obtain / diffget） │
  ├────────┼─────────────────────────────────────────┤
  │ dp     │ 推送改动到对面（diff put）              │
  ├────────┼─────────────────────────────────────────┤
  │ ]c     │ 跳到下一个冲突                          │
  ├────────┼─────────────────────────────────────────┤
  │ [c     │ 跳到上一个冲突                          │
  └────────┴─────────────────────────────────────────┘

  ---
  Diffview — Diff 视图窗口（内置默认，未覆盖的）

  ┌────────────┬───────────────────────────────┐
  │   快捷键   │             功能              │
  ├────────────┼───────────────────────────────┤
  │ <leader>co │ 冲突：选择 OURS（本地）       │
  ├────────────┼───────────────────────────────┤
  │ <leader>ct │ 冲突：选择 THEIRS（远程）     │
  ├────────────┼───────────────────────────────┤
  │ <leader>cb │ 冲突：选择 BASE（共同祖先）   │
  ├────────────┼───────────────────────────────┤
  │ <leader>ca │ 冲突：保留全部                │
  ├────────────┼───────────────────────────────┤
  │ dx         │ 冲突：删除整个冲突块（NONE）  │
  ├────────────┼───────────────────────────────┤
  │ <leader>cO │ 冲突：选择 OURS（整个文件）   │
  ├────────────┼───────────────────────────────┤
  │ <leader>cT │ 冲突：选择 THEIRS（整个文件） │
  ├────────────┼───────────────────────────────┤
  │ <leader>cB │ 冲突：选择 BASE（整个文件）   │
  ├────────────┼───────────────────────────────┤
  │ g?         │ 显示当前面板帮助              │
  └────────────┴───────────────────────────────┘

  ---
  Diffview — File Panel 文件面板（自定义）

  ┌────────┬──────────────────────┐
  │ 快捷键 │         功能         │
  ├────────┼──────────────────────┤
  │ s      │ Stage / Unstage 文件 │
  ├────────┼──────────────────────┤
  │ o      │ 打开文件 diff        │
  ├────────┼──────────────────────┤
  │ q      │ 关闭 Diffview        │
  └────────┴──────────────────────┘

  ---
  Diffview — File Panel 文件面板（内置默认，未覆盖的）

  ┌────────┬─────────────────────────────┐
  │ 快捷键 │            功能             │
  ├────────┼─────────────────────────────┤
  │ <cr>   │ 打开文件 diff（同 o）       │
  ├────────┼─────────────────────────────┤
  │ <tab>  │ 打开下一个文件 diff         │
  ├────────┼─────────────────────────────┤
  │ -      │ Stage / Unstage（同 s）     │
  ├────────┼─────────────────────────────┤
  │ S      │ Stage 全部文件              │
  ├────────┼─────────────────────────────┤
  │ U      │ Unstage 全部文件            │
  ├────────┼─────────────────────────────┤
  │ X      │ Restore 文件（丢弃改动）    │
  ├────────┼─────────────────────────────┤
  │ R      │ 刷新文件列表                │
  ├────────┼─────────────────────────────┤
  │ L      │ 打开 commit log             │
  ├────────┼─────────────────────────────┤
  │ i      │ 列表 / 树形视图切换         │
  ├────────┼─────────────────────────────┤
  │ f      │ 展平 / 展开目录             │
  ├────────┼─────────────────────────────┤
  │ gf     │ 在新分屏打开文件（非 diff） │
  ├────────┼─────────────────────────────┤
  │ g?     │ 显示帮助                    │
  └────────┴─────────────────────────────┘

  ---
  常用工作流速查

  ┌──────────────────────┬─────────────────────────────────────────────────┐
  │         场景         │                      操作                       │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 查看当前文件改了什么 │ ]h 跳变更 → <leader>gp 预览                     │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 丢弃某个 hunk        │ <leader>gr                                      │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 暂存某个 hunk        │ <leader>gs                                      │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 浏览所有改动文件     │ <leader>gd 打开 Diffview                        │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ Stage 整个文件       │ Diffview File Panel 按 s                        │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ Restore 整个文件     │ Diffview File Panel 按 X                        │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 解决冲突             │ <leader>gd → ]c 跳冲突 → <leader>co/ct 选择版本 │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 看某文件的 git 历史  │ <leader>gf                                      │
  ├──────────────────────┼─────────────────────────────────────────────────┤
  │ 退出 Diffview        │ q 或 <leader>gc                                 │
  └──────────────────────┴─────────────────────────────────────────────────┘
