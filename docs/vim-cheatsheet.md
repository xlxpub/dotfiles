## 模式切换

| 按键 | 说明 |
|------|------|
| `i` | 在光标前进入插入模式 |
| `I` | 在行首进入插入模式 |
| `a` | 在光标后进入插入模式 |
| `A` | 在行尾进入插入模式 |
| `o` | 在当前行下方新建一行并进入插入模式 |
| `O` | 在当前行上方新建一行并进入插入模式 |
| `s` | 删除当前字符并进入插入模式 |
| `S` | 删除整行并进入插入模式（等同于 `cc`） |
| `v` | 进入字符可视模式 |
| `V` | 进入行可视模式 |
| `Ctrl-v` | 进入块可视模式 |
| `R` | 进入替换模式（覆盖字符） |
| `Esc` / `Ctrl-[` | 返回普通模式 |
| `:` | 进入命令行模式 |

## 光标移动

### 基础移动

| 按键 | 说明 |
|------|------|
| `h` `j` `k` `l` | 左 / 下 / 上 / 右 |
| `0` | 行首（绝对） |
| `^` | 行首（第一个非空白字符） |
| `$` | 行尾 |
| `g_` | 行尾（最后一个非空白字符） |
| `gg` | 文件第一行 |
| `G` | 文件最后一行 |
| `:{n}` 或 `{n}G` | 跳到第 n 行 |

### 单词级移动

| 按键 | 说明 |
|------|------|
| `w` | 移到下一个单词词首 |
| `b` | 移到上一个单词词首 |
| `e` | 移到当前/下一个单词词尾 |
| `W` `B` `E` | 同上，但以空白符为分隔（WORD） |
| `ge` | 移到上一个单词词尾 |

### 屏幕级移动

| 按键 | 说明 |
|------|------|
| `H` | 移到屏幕顶部 |
| `M` | 移到屏幕中部 |
| `L` | 移到屏幕底部 |
| `Ctrl-f` | 向下翻一页 |
| `Ctrl-b` | 向上翻一页 |
| `Ctrl-d` | 向下翻半页 |
| `Ctrl-u` | 向上翻半页 |
| `zt` | 当前行滚动到屏幕顶部 |
| `zz` | 当前行滚动到屏幕中央 |
| `zb` | 当前行滚动到屏幕底部 |

### 字符跳转

| 按键 | 说明 |
|------|------|
| `f{char}` | 向右跳到字符 char（光标落在字符上） |
| `F{char}` | 向左跳到字符 char |

| `T{char}` | 向左跳到字符 char 后一位 |
| `;` | 重复上次 f/F/t/T |

## 文本对象

文本对象与操作符（`d`、`c`、`y`、`v`）结合使用，格式为：`{操作符} i/a {对象}`

- `i`（inner）= 仅内容，不含边界
- `a`（around）= 含边界（括号、引号等）

| 对象 | 示例 | 说明 |
|------|------|------|
| `w` | `diw` | 单词 |
| `W` | `daW` | WORD（以空格分隔） |
| `s` | `dis` | 句子 |
| `p` | `dip` | 段落 |
| `"` | `ci"` | 双引号内 |
| `'` | `ca'` | 单引号（含引号） |
| `` ` `` | ``ci` `` | 反引号内 |
| `(` / `)` | `di(` | 圆括号内 |
| `[` / `]` | `da[` | 方括号（含括号） |
| `{` / `}` | `ci{` | 花括号内 |
| `<` / `>` | `da<` | 尖括号（含括号） |
| `t` | `cit` | HTML/XML 标签内 |

## 查找与替换

### 查找

| 按键 | 说明 |
|------|------|
| `/{pattern}` | 向下搜索 |
| `?{pattern}` | 向上搜索 |
| `n` | 下一个匹配 |
| `N` | 上一个匹配 |
| `*` | 搜索光标下的单词（向下） |
| `#` | 搜索光标下的单词（向上） |
| `g*` / `g#` | 同上，但允许部分匹配 |

> 搜索时末尾加 `\c` 忽略大小写，如 `:/foo\c`

### 替换（命令行模式）

```
# 当前行
:s/old/new/         # 替换第一个
:s/old/new/g        # 替换所有

# 全文
:%s/old/new/g       # 全文替换
:%s/old/new/gc      # 全文替换（逐个确认）
:%s/old/new/gi      # 全文替换（忽略大小写）

# 指定行范围
:10,20s/old/new/g   # 第 10~20 行替换

# 可视模式选中后替换
:'<,'>s/old/new/g
```

## 可视模式

| 按键 | 说明 |
|------|------|
| `v` | 字符可视模式 |
| `V` | 行可视模式 |
| `Ctrl-v` | 块可视模式 |

### 块可视模式技巧

```
Ctrl-v → 选列 → I → 输入 → Esc   # 多行同时插入前缀
Ctrl-v → 选列 → A → 输入 → Esc   # 多行同时追加后缀
Ctrl-v → 选列 → d                 # 删除多列
Ctrl-v → 选列 → r{char}           # 替换多列字符
```

## 宏

| 按键 | 说明 |
|------|------|
| `q{a-z}` | 开始录制宏到寄存器 a-z |
| `q` | 停止录制 |
| `@{a-z}` | 执行寄存器中的宏 |
| `@@` | 重复上次执行的宏 |
| `{n}@{a-z}` | 执行宏 n 次 |

### 宏使用示例

```
# 给每行末尾加分号
qq          # 开始录制到寄存器 q
A;          # 跳到行尾插入分号
j           # 移到下一行
Esc         # 退出插入模式
q           # 停止录制
100@q       # 对接下来 100 行执行该宏
```

## 实用技巧

### 数字前缀

几乎所有命令都可加数字前缀 `{n}` 表示重复次数：

```
5j      # 向下移动 5 行
3dw     # 删除 3 个单词
2yy     # 复制 2 行
10@q    # 执行宏 10 次
```

### 高频组合操作

```
ci"     # 修改引号内容（最常用！）
da{     # 删除花括号及其内容
yap     # 复制整个段落
=G      # 自动缩进到文件末尾
ggVG    # 全选
ggdG    # 清空文件
:%y+    # 复制全文到系统剪贴板（macOS）
```

### 插入模式下的快捷键

| 按键 | 说明 |
|------|------|
| `Ctrl-w` | 删除光标前一个单词 |
| `Ctrl-u` | 删除光标前到行首 |
| `Ctrl-r "` | 粘贴默认寄存器内容 |
| `Ctrl-r =` | 插入计算结果（如 `:r=2+3` 插入 5） |
| `Ctrl-o {cmd}` | 临时执行一条普通模式命令后返回插入模式 |
| `Ctrl-n/p` | 基于文件内容的词语补全 |

### 查找时的正则表达式

```
/\bword\b   # 精确匹配单词
/^pattern   # 行首
/pattern$   # 行尾
/\d\+       # 一个或多个数字
/\v(foo|bar) # 使用 \v 简化正则（very magic 模式）
```

### 外部命令过滤

```
:!python3 %         # 运行当前文件
:%!jq .             # 用 jq 格式化整个 JSON 文件
:'<,'>!sort         # 对选中行排序（调用系统 sort）
```
